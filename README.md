EXCEL PORTFOLIO PROJECT – BIKE BUYERS ANALYSIS

PROJECT OVERVIEW

* This project focuses on analyzing bike buyer data using Microsoft Excel to understand customer demographics, income patterns, and purchasing behavior. The goal is to convert raw data into meaningful insights through data cleaning and interactive dashboards.

DATASET

* The project uses a bike buyers dataset containing customer details such as age, gender, marital status, education, region, income, commute distance, and purchase status.
* Both the original dataset and the cleaned version are included in this repository.

DATA CLEANING

* Data cleaning was performed in Excel to improve accuracy and usability. This includes handling inconsistencies, creating age brackets,
  standardizing categorical values, and preparing the dataset for analysis and visualization.

DASHBOARD

* An interactive Excel dashboard was built to visualize key insights such as:

* Average income by gender and purchase status

* Customer age brackets vs bike purchases

* Commute distance and purchase trends

* Filters for region, education, marital status, and gender

* A dashboard preview image is included in the repository.

TOOLS USED

* Microsoft Excel

* Pivot Tables

* Pivot Charts

* Slicers

* Basic Data Cleaning Techniques

FILES IN THIS REPOSITORY

KEY LEARNINGS
